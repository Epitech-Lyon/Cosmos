# Cosmos

Another x86 kernel named COSmOS.

## Dependencies

* `make` (dev under v4.3)
* `grub-mkrescue` (dev under v2.04)
* `gcc` (dev under v10.1.0)

## Build

`make`

### Kernel build

`make kernel`

output an *elf* file.

### ISO build

`make iso`

output an *iso* file.

## Run

A configured *QEMU* run is provided.

`make run`
